Política de Privacidade

Última atualização: 07 de abril de 2025

Este aplicativo respeita a sua privacidade. Abaixo explicamos como lidamos com os dados do usuário.

1. Coleta de Informações

O aplicativo coleta apenas o nome do usuário, com o único propósito de personalizar a experiência e exibir o nome na interface do app.

2. Armazenamento de Dados

Todos os dados inseridos no aplicativo, incluindo sonhos e outras informações pessoais, são armazenados localmente no dispositivo do usuário. Nenhum dado é enviado para servidores externos ou compartilhado com terceiros.

3. Compartilhamento de Dados

Não compartilhamos nenhum dado com terceiros. Como o app funciona de forma totalmente offline, os dados permanecem sob controle exclusivo do usuário.

4. Segurança

Por funcionar localmente, o aplicativo depende da segurança do próprio dispositivo do usuário (como bloqueio de tela) para proteger os dados armazenados. Recomendamos que o usuário mantenha seu dispositivo seguro.

5. Direitos do Usuário

O usuário tem total controle sobre suas informações. Para excluir seus dados, basta removê-los manualmente no próprio app ou desinstalá-lo do dispositivo.

6. Alterações nesta Política

Podemos atualizar esta política periodicamente. Recomendamos revisar esta página ocasionalmente para se manter informado sobre qualquer mudança.
